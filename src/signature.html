Comment installer votre signature ?

Google Workspace / Gmail

Gmail n'a pas de bouton HTML. Vous devez copier le résultat visuel depuis votre navigateur.

  1. Ouvrez ce fichier dans votre navigateur (Chrome, Safari...).
  2. Faites Cmd + A (Tout sélectionner) puis Cmd + C (Copier).
  3. Dans Gmail : Paramètres > Voir tous les paramètres > Signature.
  4. Cliquez sur + Créer, nommez-la, et faites Coller (Cmd + V) dans la zone de droite.
  5. Configuration : Dans "Paramètres par défaut", sélectionnez votre nouvelle signature pour les nouveaux e-mails et réponses.
  6. Cochez "Insérer la signature avant le texte des messages précédents..." si souhaité.
  7. Cliquez sur Enregistrer les modifications tout en bas.
  8. Si l'image est brisée : Utilisez le bouton "Insérer une image" > "Adresse Web (URL)" et collez l'URL de votre logo hosté (ex: https://kerlinkin.netlify.app/assets/images/logo.png).

Apple Mail (Mac)

Apple Mail ne permet pas de coller directement du HTML. Il faut remplacer un fichier de signature existant.

  1. Dans Mail > Réglages > Signatures, créez une nouvelle signature "fictive" (ex: "HTML Signature") avec un texte simple.
  2. Quittez complètement "Mail" (Cmd+Q). C'est crucial.
  3. Dans le Finder, allez à ~/Library/Mail/ (via le menu "Aller > Aller au dossier").
  4. Ouvrez le dossier V10 (ou le chiffre le plus élevé) > MailData > Signatures.
  5. Trouvez le fichier .mailsignature le plus récent et ouvrez-le avec TextEdit.
  6. Gardez les premières lignes d'en-tête (MIME-Version, etc.) et remplacez tout le contenu en dessous par votre code HTML.
  7. Sauvegardez, puis faites Lire les informations (Cmd+I) sur le fichier et cochez "Verrouillé" pour empêcher Mail de l'écraser.
  8. Rouvrez Mail.

iPhone (iOS Mail)

Vous devez copier le rendu visuel et utiliser une astuce pour préserver le formatage.

  1. Ouvrez cette page sur votre iPhone (ou envoyez-vous la signature par email) et copiez tout le rendu visuel (pas le code HTML).
  2. Allez dans Réglages > Mail > Signature.
  3. Collez la signature dans la zone de texte.
  4. Si le formatage casse : Secouez immédiatement votre iPhone pour faire apparaître "Annuler les attributs" et cliquez sur Annuler. Cela restaurera le style original.
